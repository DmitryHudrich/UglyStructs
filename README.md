# [Ru] UglyStructs
### - это мой проект для изучения простецких базовых штук, связанных с алгоритмами, структурами данных и чем-то еще из этой темы в рамках изучения языка программирования C++.

## Структуры данных:
### Реализовано:
* Ничего хаха;
### Частично реализовано:
* Динамический массив;
### В планах:
* Массив (Надо ли?);
* Связный и двусвязный список;
* Стек основанный на массиве и стек основанный на узлах;
* Очередь основанная на узлах;
* Множество;
* Хеш-таблица;
#### А также деревья:
* Двоичное дерево поиска;
* Красно-черное дерево;
* Префиксное дерево;
* (Другие виды деревьев будут добавляться в будущем);
## Алгоритмы:
### Реализовано:
* Ничего хаха;
### Частично реализовано:
* Ниче;
### В планах:
#### Сортировка:
* Пузырьковая сортировка;
* Сортировка выбором;
* Сортировка вставками;
* Быстрая сортировка;
* Сортировка слиянием;
* Сортировка Шелла;
* Сортировка кучей;
#### Нахождение пути:
* Алгоритм A*;
* Алгоритм Дейкстры;
#### А также виды алгоритмов, которые пока не вошли в классификацию:
* Нахождение последовательности;
* Математические расчеты;

Списки будут дополняться.
